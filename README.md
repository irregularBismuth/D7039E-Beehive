# Board
NUCLEO-L073RZ NUCLEO development board

## Sensor card
STM32L073RZT6, hal: https://crates.io/crates/stm32l0
ARM Cortex M0+
Embedded ST-LINK V2 debugger and programmer

## Gateway card
STM32F746ZGT6, hal: https://github.com/stm32-rs/stm32f7xx-hal
ARM Cortex M7
Embedded ST-LINK V2 debugger and programmer
